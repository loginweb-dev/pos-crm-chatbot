## Deployment

- git clone https://github.com/loginweb-dev/pos-crm-chatbot
- composer install
- cp .env.example .env
- php artisan storage:link
- php artisan key:generate
- chmod -R 777 storage bootstrap/cache
- npm install (public/chatbot)
